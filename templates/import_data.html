📥 Import Historical Data (Jan-March 2025)

Import customers and tasks from Excel files with original dates preserved

Step 1: Import Customers

Required columns: Name, Company, Phone, Email, Assigned To, Customer Type
Excel Template Format:
Column A: Customer Name
Column B: Company
Column C: Phone
Column D: Email
Column E: Assigned To (staff name)
Column F: Customer Type (Individual/Company)

Step 2: Import Tasks

Required columns: Customer Name, Job Type, Status, Created Date, Amount Invoiced, Amount Received
Excel Template Format:
Column A: Customer Name
Column B: Job Type
Column C: Status
Column D: Created Date (DD/MM/YYYY)
Column E: Due Date (DD/MM/YYYY)
Column F: Completed Date (DD/MM/YYYY)
Column G: Assigned To (staff name)
Column H: Invoiced (AED)
Column I: Received (AED)
Column J: Priority
Column K: Remarks

⚠️ Important Instructions

  1. Import customers FIRST, then import tasks (tasks need customers to exist)
  2. Staff names in Excel must match existing staff names in the system
  3. Customer names in tasks file must exactly match customer names from Step 1
  4. Dates will be preserved as-is from your Excel (historical dates maintained)
  5. Review preview carefully before confirming import
  6. This is a ONE-TIME import - after completion, this page will be removed
